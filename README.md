# LinkedIn for Businesses

A B2B networking and investment marketplace platform built with Payload CMS (backend) and Next.js (frontend).

## Monorepo Structure
```
apps/
  backend/   # Payload CMS server
  frontend/  # Next.js app
packages/
  shared/    # Shared utilities/types
scripts/
  clean.ts   # Clean script
```
